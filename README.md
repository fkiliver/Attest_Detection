# Artifact Release

This directory contains the artifact package for the paper.

## Contents

- `method/`: sanitized implementations of the proposed method and related method variants.
- `baseline_reproduction_methods/`: baseline reproduction scripts and third-party source snapshots.
- `hardset_case_studies/`: curated hard-case datasets, case studies, and manifests.

## Quick Start

See the README, requirements, and run scripts inside each subdirectory for component-specific instructions. The hard-case package includes `hardset_case_studies/MANIFEST.json` and `hardset_case_studies/MANIFEST.tsv` for file inventory and hashes.
